SLYR
Research Architecture Roadmap

A real-time autonomous avionics system for UAVs.

SLYR is a research-driven system focused on combining deterministic control with intelligent decision-making.

The Problem

Most UAV systems tightly couple flight control with high-level logic, resulting in unstable behavior, poor scalability, and limited autonomy. AI systems cannot be directly trusted with control loops due to latency and unpredictability.

Architecture

SLYR uses a dual-layer architecture separating real-time control from intelligent autonomy. 

[Sensors]
   ↓
[STM32 Flight Kernel]
 - Sensor Fusion
 - PID Control
 - Motor Output
   ↓
[Communication Layer]
   ↓
[Autonomy Layer (Pi)]
 - Navigation
 - Perception
 - Decision Engine (SLM)
   ↓
[Command Shaping]
   ↓
[Control Targets → STM32]

Roadmap (7 Months)

Month 1: Communication layer + STM32 bring-up
Month 2: Basic stabilization (IMU + PID)
Month 3: Attitude + altitude hold
Month 4: GPS navigation + waypoint system
Month 5: Obstacle detection (LiDAR/Radar)
Month 6: SLM integration + behavior system
Month 7: Full mission autonomy + testing
